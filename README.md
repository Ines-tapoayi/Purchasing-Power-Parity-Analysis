# Purchasing-Power-Parity-Analysis
## Overview
This project investigates the validity of Purchasing Power Parity (PPP) for several Eurozone countries against the US Dollar over the 2002–2025 period.

The study combines:
- real exchange rate analysis
- unit root testing
- structural break detection
- long-run econometric modeling
- exchange rate risk analysis

## Countries Studied
- France
- Germany
- Italy
- Belgium
- Austria

## Objectives
- Test the validity of PPP
- Analyze long-term exchange rate dynamics
- Detect structural breaks after 2015
- Study the role of exchange rate risk and uncertainty

## Methodology
### Standard PPP Tests
- ADF unit root tests
- Real exchange rate stationarity analysis
- Long-run OLS regressions

### Structural Break Analysis
- Chow structural break test
- Pre-2015 vs Post-2015 comparison

### Risk-Based Extensions
Following Arghyrou et al. (2025):
- Variance
- Skewness
- Kurtosis

were integrated to explain persistent PPP deviations.

## Key Findings
- Absolute PPP is generally rejected
- Strong structural break detected around 2015
- Exchange rate risk significantly affects real exchange rate dynamics
- PPP behaves more as a long-run equilibrium reference than a strict law
